## 项目简介

AI 八字算命助手是一个前后端分离的 Web 应用，围绕“命盘解读 + 今日日运 + AI 陪伴式问答”构建完整体验。用户可以通过邮箱验证码登录，创建个人命盘档案，查看命盘解读结果和每日运势，并结合当天上下文进行 AI 提问，同时还可以查看成长档案和管理提醒设置。

## 技术栈

### Frontend
- Vue 3
- TypeScript
- Vite
- Vue Router
- Pinia
- Axios

### Backend
- FastAPI
- Pydantic
- Uvicorn

## 功能特性

- 邮箱验证码登录
- 用户命盘档案创建与保存
- 命盘解读结果展示
- 今日日运查询与历史查看
- AI 问答交互
- 成长档案聚合展示
- 提醒设置管理
- 统一 API 响应结构
- Token 鉴权机制
- 本地联调支持（Axios + Vite Proxy / CORS）

## 项目特点

- 前后端分离，结构清晰，便于扩展
- 前端采用模块化状态管理与接口封装
- 后端采用 FastAPI 提供统一 REST API
- 支持本地开发环境联调
- 具备完整的基础业务闭环：登录、建档、解读、查询、问答、提醒
